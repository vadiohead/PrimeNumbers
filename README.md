# Prime Numbers
### 🇲🇳 Буряад-Монгол
Android 7.1.1 болон шэнэ түхеэрэлгэнүүдтэ Android Studio Meerkat дээрэ бүтээгдэһэн. 7 хэлэн дээрэ бии. Татажа абаха APK файл болон эхинэй кодые "Releases" хэһэгтэ шалгагты.

### 🇬🇧 English
Built on Android Studio Meerkat for Android 7.1.1 devices and newer. Available in 7 languages. Check "Releases" section for downloadable APK file and source code.
